NovaStrike

Godot 4.6.1 · GDScript

A 2D vertical shooter with wave-based difficulty scaling. Features hand-drawn soldiers, zombies, and demons using Godot's _draw() API with real-time animation. Includes procedural WAV audio, 3-layer parallax starfield with twinkling, screen shake, particle explosions, and a multi-track BGM system that intensifies with each wave.
